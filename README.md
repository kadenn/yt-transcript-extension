# YT Transcript Copier

A minimal Chrome extension to copy YouTube video transcripts with one click.

## Features

- **Copy Transcript** — copies the full transcript as plain text
- **Copy with Timestamps** — copies with `[0:34]` timestamps per line
- Works with any language YouTube supports

## Install

1. Clone or download this repo
2. Go to `chrome://extensions` and enable **Developer mode**
3. Click **Load unpacked** and select the folder

## Usage

Open any YouTube video → click the extension icon → hit Copy.

> If the button is greyed out, make sure the video has captions available.
